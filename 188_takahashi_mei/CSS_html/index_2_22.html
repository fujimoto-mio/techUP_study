フランスパン

フランスパンの起源と由来

「フランスパン」は、日本で広く普及する際に「フランス風のパン」として定着した名称です。 明治時代以降、西洋文化が日本に入り始めた頃にフランス式の硬いパンを紹介され、そう呼ばれるようになりました。

本場フランスでの歴史

フランスでのフランスパンの原型は、18世紀末から19世紀にかけてすでに存在していました。 特に現在の「バゲット」の形が定着したのは、1920年代以降といわれています。

この頃、労働法でパン職人の出勤時間が遅くなり、早く焼ける細長いパンが求められるようになって、バゲットが広まったとされます。

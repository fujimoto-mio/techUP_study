About

Juitaについて

Juita Web Designは、インドネシア（ジャカルタ含む） バリ島にてホームページ制作業務を中心にフリーランスとして活動しているWebデザイナー事務所（SOHO）。

デザイナーは、大学（日本語学科）を卒業後、日本へ留学。
その後、Webデザイン専門学校（神奈川県横浜市）へ入学し日本のWeb知識、スキルを2年間学ぶ。Webデザイン専門学校卒業後は、アパレル会社（神奈川県藤沢市）にWebデザイナーとして勤務。 その傍ら、日本で起業、開店する外国人を主にWebデザイナーとしてサポート。その後、出版会社（東京都千代田区）にWebデザイナーとして勤務、ジャカルタ支社たち上げの為、支社長としてジャカルタへ赴任。その後、バリ島にてJuita Web Designをたち上げる。

Q&A

よくある質問

Q 日本からのホームページ制作の依頼は可能ですか？

A承っております。Juitaに頂くご依頼の多くは、日本などインドネシア国外からです。まずはお気軽にご相談ください。
メールやりとりだけでなく、Skypeでの打ち合わせ、実際にお会いしての面談なども可能でございます。（バリ島以外での面談の場合は交通費別途）

Q メールやSkype打ち合わせなどは日本語で可能ですか？

A すべて日本語で対応しております。ご安心ください。

Q ホームページ掲載原稿をインドネシア語から日本語へ翻訳も依頼できますか？

A 承っております。（日本語⇔インドネシア語）別途、翻訳料金がかかります。

Q 事前に必要なものはありますか？

Aホームページに掲載する原稿や写真、イラストなどのデータをご用意ください。 ご提出いただいた原稿をもとに制作を進めてまいります。

Q ドメインやサーバーの手配も依頼できますか？

A 承っております。お客様のご要望に合わせた最適なサーバープランをご提案させていただきます。

Q HTMLなどの知識が無いのですが、自分で更新できますか？

A 頻繁に更新するようなページをご自身で更新されたい場合は、CMSの導入をお勧めしております。CMSを導入することで、ブラウザ上から手軽に更新することが可能となります。
また、お客様ご自身では難しい更新内容等ありましたら、更新の代行やレッスンも承っておりますので、お気軽にご相談ください。

Q 納品後のサポートもお願いできますか？

A 契約期間の縛りはございません。月単位でのご契約が可能です。

Q サービスの契約期間はありますか？

A 承っております。ホームページ公開後の更新業務や管理など、ご不明な点やご相談等ありましたら、お気軽にお問い合わせください。

Q クレジットカード・小切手等での支払いはできますか？

A 申し訳ありませんが、お支払いは現金と銀行振込のみとさせていただいております。お支払いの際に、日本口座または、インドネシア口座をお伝えいたします。
